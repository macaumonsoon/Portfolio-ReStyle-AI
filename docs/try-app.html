Run the Portfolio ReStyle AI app 運行 Portfolio ReStyle AI 工具

This GitHub Pages site is static. The full interactive tool is a Next.js app that runs on your computer (Node.js 20+). Follow the steps below, then open the local URL.

GitHub Pages 上的是靜態介紹站，完整的互動工具是需在您電腦上運行的 Next.js 應用（需 Node.js 20+）。請依下列步驟操作，再在瀏覽器開啟本機網址。 

git clone https://github.com/macaumonsoon/Portfolio-ReStyle-AI.git
cd Portfolio-ReStyle-AI/web
npm install
npm run dev
Open app at http://127.0.0.1:3000 → 開啟本機應用 http://127.0.0.1:3000 →
Full setup on GitHub README → GitHub README 完整說明 → Watch demo video → 觀看演示影片 →

Use the purple button only after npm run dev is running. If the page does not load, check the terminal for errors or a different port.

請在終端機已成功執行 npm run dev 之後再點紫色按鈕。若無法開啟，請查看終端機是否報錯或使用其他埠號。
