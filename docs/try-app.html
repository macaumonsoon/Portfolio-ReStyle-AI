Try Portfolio ReStyle AI 試用 Portfolio ReStyle AI

In your browser (Vercel) 瀏覽器線上試用（Vercel）

After deploy, no install is needed in the browser. Import this GitHub repo on Vercel, set Root Directory to web, deploy once, then paste the Production URL into docs/assets/prsa-config.js (PRSA_PRODUCTION_APP_URL) and push.

部署後訪客無需安裝 Node。請在 Vercel 導入本 GitHub 倉庫，Root Directory 設為 web，部署完成後把 Production 網址寫入 docs/assets/prsa-config.js 的 PRSA_PRODUCTION_APP_URL 並推送。

Deploy on Vercel (root: web) → 在 Vercel 部署（根目錄選 web）→ Open live app → 開啟線上應用 → Open local app (after npm run dev) → 開啟本機應用（先 npm run dev）→
How to set Root Directory → 如何設定根目錄 web → Watch demo video → 觀看演示影片 →

While PRSA_PRODUCTION_APP_URL is empty, the purple button opens Vercel import (not a 404). After your first successful deploy, paste the Production URL into docs/assets/prsa-config.js and push; the button label switches to “Open live app”.

PRSA_PRODUCTION_APP_URL 留空時，紫色按鈕會打開 Vercel 導入流程（不會再進入無效部署 404）。首次部署成功後，把 Production 網址貼入 docs/assets/prsa-config.js 並推送，按鈕會變為「開啟線上應用」。

Or locally 或本機運行

On your computer (Node.js) 在您的電腦上（Node.js）

The full interactive tool is a Next.js app. Requires Node.js 20+. Clone the repo, run the commands below, then open the local URL.

完整互動工具為 Next.js 應用，需 Node.js 20+。複製倉庫後依下列指令執行，再在瀏覽器開啟本機網址。 

git clone https://github.com/macaumonsoon/Portfolio-ReStyle-AI.git
cd Portfolio-ReStyle-AI/web
npm install
npm run dev
Open local app at http://127.0.0.1:3000 → 開啟本機應用 http://127.0.0.1:3000 →
Full setup on GitHub README → GitHub README 完整說明 →

Use the local purple button only after npm run dev is running. If the page does not load, check the terminal for errors or a different port.

請在終端機已成功執行 npm run dev 之後再點本機紫色按鈕。若無法開啟，請查看終端機是否報錯或使用其他埠號。
