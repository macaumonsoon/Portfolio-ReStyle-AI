**Background**

With the rapid development of artificial intelligence and generative technologies, the process of producing design portfolios is gradually shifting from manual layout toward intelligent assistance. For designers, a portfolio is not only a collection of visual works but also a structured narrative that communicates design thinking, project processes, and professional identity. However, creating a well-structured portfolio often requires significant time spent on repetitive layout adjustments, grid alignment, image scaling, and typographic organization. These tasks, while essential for visual consistency, can distract designers from focusing on the storytelling and conceptual aspects of their work.

Existing design tools primarily provide manual layout systems such as grids, guides, and templates. While these tools offer flexibility, they still rely heavily on the designer’s time and technical expertise to organize content effectively. As portfolios grow in complexity—often including multiple projects, images, diagrams, and explanatory texts—the process of restructuring or redesigning an existing portfolio becomes labor-intensive.

Recent advances in AI-driven design tools suggest new possibilities for automating parts of the design process. Computer vision models can analyze visual content, extract layout structures, and identify relationships between images and text, while generative models can propose alternative layouts based on design principles such as hierarchy, balance, and grid systems. Despite these technological developments, most current tools still lack the ability to understand the narrative logic of a portfolio and assist designers in restructuring it holistically.

This project proposes an AI-assisted web-based design tool that helps designers reinterpret and reorganize their existing portfolios. By allowing users to upload portfolio files (such as PDF or SVG), the system analyzes the visual and textual content, decomposes the layout into modular elements, and reconstructs new page compositions based on grid systems, stylistic preferences, and narrative structures specified by the user. Instead of producing a single automated result, the system generates multiple layout variations for each page, enabling designers to iteratively select and refine their preferred version.

In addition to automated layout generation, the project explores new forms of interaction between designers and AI systems. Through technologies such as real-time preview, spatial visualization, and interactive manipulation, the tool aims to transform portfolio creation from a static editing process into a collaborative design experience between human designers and intelligent systems.

Ultimately, this project investigates how artificial intelligence can support designers not merely as an automation tool but as a creative collaborator that helps restructure visual narratives, explore alternative layouts, and accelerate the process of portfolio design.
